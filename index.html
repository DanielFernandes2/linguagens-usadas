O Arduino é baseado na linguagem C++. Para compilar um programa é necessário ter um ambiente de desenvolvimento, neste caso, o Arduino IDE que é um aplicativo de computador com compilador integrado. Para testar o Arduino, a placa deve estar conectada via USB no computador onde a um terminal que o código está escrito e nele será compilado e gravado na placa.

O Arduino é uma placa que aceita códigos pequenos, como ligar um Led em um determinado tempo, ou ate mesmo usado em códigos mais complexos como por exemplo controlar sensores de diversos tipos.

Durante o ano fizemos vários projetos com o Arduino, como um conjunto de sensores para ter manutenção preventiva de maquinas que estão tremendo muito, outro foi o projeto final do ano junto com a empresa DOW que fizemos uma tomada inteligente usando sensores.

python.py
O Python é uma linguagem de programação amplamente usada em aplicações da Web, desenvolvimento de software, ciência de dados e machine learning(ML). Os desenvolvedores usam o Python porque é eficiente e fácil de aprender e pode ser executada em muitas plataformas diferentes.

A linguagem oferece recursos como tipagem dinâmica e forte (tipo de dado do valor deve ser do mesmo tipo da variável), orientação a objetos, multi-paradigmas (programação funcional e imperativa), além de recursos poderosos em biblioteca padrão e via módulos e frameworks desenvolvidos pela comunidade. Seu código é aberto e a utilização é gratuita, rodando em praticamente qualquer sistema operacional.

Durante o ano utilizamos o Python para fazer alguns códigos com temas específicos por exemplo notas de uma escola, para ver se o aluno bateu a media escolar.
